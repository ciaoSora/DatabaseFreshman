# 数据库项目自动AC机

此项目系鸭大2018级软工教务2班，大一秋季学期的项目。此自动AC机使用Python 3.x写就。

## 所需三方库

- requests

## 功能

- 简单的操作
- ~~一刀999~~ 一键做题

## 使用方法

- 修改 solution/config.json 文件，改为自己的账号、密码
- 双击运行 solution/get_ans.py，静候预处理
- 双击运行 solution/Yi_Jian_A_Ti.py，输入想做的题数，即可**自动**做题

## 注意事项

- 运行脚本后，浏览器登陆的账号会被顶掉
- get_ans.py 运行时，会 **“智能” 放弃**一些题，一般在五题以下，所以不必过于担心。放弃题目时，终端会显示当前放弃题目数量。当你认为放弃题目过多，不信任该脚本，可以随时关闭脚本。
